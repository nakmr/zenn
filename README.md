# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)
* [ZennのMarkdown記法一覧](https://zenn.dev/zenn/articles/markdown-guide)

```bash
# 新しい記事を書く
npx zenn new:article

# 新しい本を書く
npx zenn new:book

# プレビュー
npx zenn preview

# textlint
npx textlint articles/hoge.md
npx textlint --fix articles/hoge.md
```
